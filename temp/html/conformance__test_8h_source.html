157  // Whether to require the testee to pass RECOMMENDED tests. By default failing
158  // a RECOMMENDED test case will not fail the entire suite but will only
159  // generated a warning. If this flag is set to true, RECOMMENDED tests will
160  // be treated the same way as REQUIRED tests and failing a RECOMMENDED test
161  // case will cause the entire test suite to fail as well. An implementation
162  // can enable this if it wants to be strictly conforming to protobuf spec.
163  // See the comments about ConformanceLevel below to learn more about the
164  // difference between REQUIRED and RECOMMENDED test cases.
165  void SetEnforceRecommended(bool value) {
166  enforce_recommended_ = value;
167  }

182  // Run all the conformance tests against the given test runner.
183  // Test output will be stored in "output".
184  //
185  // Returns true if the set of failing tests was exactly the same as the
186  // failure list.
187  // The filename here is *only* used to create/format useful error messages for
188  // how to update the failure list. We do NOT read this file at all.
189  bool RunSuite(ConformanceTestRunner* runner, std::string* output,
190  const std::string& filename,
191  conformance::FailureSet* failure_list);
192 
193  protected:
194  // Test cases are classified into a few categories:
195  // REQUIRED: the test case must be passed for an implementation to be
196  // interoperable with other implementations. For example, a
197  // parser implementation must accept both packed and unpacked
198  // form of repeated primitive fields.
199  // RECOMMENDED: the test case is not required for the implementation to
200  // be interoperable with other implementations, but is
201  // recommended for best performance and compatibility. For
202  // example, a proto3 serializer should serialize repeated
203  // primitive fields in packed form, but an implementation
204  // failing to do so will still be able to communicate with
205  // other implementations.
206  enum ConformanceLevel {
207  REQUIRED = 0,
208  RECOMMENDED = 1,
209  };
